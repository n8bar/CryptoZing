# site/

This is the temporary GitHub Pages placeholder, **not** the Laravel app.

- Treat it as a separate static surface even though it lives in the same repo.
- Do not pull Laravel routes, Blade components, app config, or `vendor/` references into files here.
- Authoring is plain HTML / Markdown / Jekyll-style; `site/_layouts/` and `site/learn/` follow GitHub Pages conventions.
- Staging copy under `site/staging/` should be edited before the published copy under `site/` (see memory: "Update Staging Copy First").
