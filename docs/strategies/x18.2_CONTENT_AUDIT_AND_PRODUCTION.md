Status: Complete.
Parent milestone doc: [`docs/milestones/18_PRERELEASE_CONTENT_SEO.md`](../milestones/18_PRERELEASE_CONTENT_SEO.md)

**Goal:** Audit the existing Helpful Notes, make Adapt/Inspire/Skip decisions, and produce all planned content — moving directly from audit decision to writing for each piece.

## Publish strategy

Content and doc changes go straight to `main`. Branches and PRs are reserved for code changes. GitHub Pages deploys from `main`, so pushing content to `main` is the publish step.

## Content workflow

1. **Research** — investigate the topic before writing; document findings in `docs/research/<slug>.md` as internal notes (not published). These notes are the raw material for the article and include, but are not limited to: competitor/tool landscape, feature comparisons, claims we can verify vs claims we're taking at face value, and open questions.
2. **Author** — create `site/staging/<slug>.md` with front matter: `layout: article.njk`, `title`, `description`, `canonical: https://cryptozing.app/learn/<slug>/` (the future production URL)
3. **Build** — `cd site && npx @11ty/eleventy` — or run `npx @11ty/eleventy --watch` once and skip this step on every subsequent save
4. **Stage** — review locally at `http://<dev-server-LAN>/staging/<slug>/` via Sail (no push needed), or push to `main` for a live shareable staging URL at `https://cryptozing.app/staging/<slug>/` — noindexed, not linked from anywhere
5. **Review** — read it, revise, repeat until satisfied; update [`docs/CONTENT_PROMISES.md`](../CONTENT_PROMISES.md) with any new promises
6. **Publish** — copy file from `site/staging/<slug>.md` to `site/learn/<slug>.md`; push to `main`. The staging copy stays as the working copy for future edits — when changes are ready, copy to learn again and push.
7. **Verify** — confirm HTTP 200 at `https://cryptozing.app/learn/<slug>/`
8. **Sitemap** — add the URL to `site/sitemap.xml` with today's date as `lastmod`
9. **Index** — IndexNow pings Bing/Yandex/Naver automatically on deploy; Claude outputs the production URL for copy-paste into Google Search Console → URL Inspection → Request Indexing
